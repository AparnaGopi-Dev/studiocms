# StudioCMS

Photography Portfolio CMS built with Laravel.

## Features

✓ Admin dashboard
✓ Portfolio management
✓ Photo albums
✓ Booking enquiries
✓ Contact management
✓ Responsive luxury photography website

## Tech Stack

Laravel 13
PHP 8.3
MySQL
Tailwind CSS
Vite

## Installation

composer install
npm install
php artisan migrate
npm run dev
